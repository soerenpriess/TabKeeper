## Overview

Organize your open tabs as clear tab sets! Save, group, and name tab collections with custom colors and icons. Switch between workspaces, open tabs with one click, and boost your productivity 

**Why TabKeeper?**

This project addresses the common pain points of tab clutter and disorganization. The core features include:

- üé® **Dynamic Tab Management:** Easily save, delete, and organize tabs to reduce clutter.
- üîÑ **One-Click Workspace Switching:** Switch between different tab sets with a single click. Perfect for separating work, study, and personal browsing sessions without losing your place.
- üõ†Ô∏è **Customizable Tab Sets:** Personalize your tab sets with colors and icons for better visual organization.
- üñ•Ô∏è **User-Friendly Interface:** Intuitive components enhance user interaction and overall experience.
- üíæ **Local Storage Integration:** Efficiently manage tab sets with persistent storage for easy access.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** Vue.js
- **Package Manager:** Npm

### Installation

Build TabKeeper from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ‚ùØ git clone https://github.com/soerenpriess/TabKeeper
    ```

2. **Navigate to the project directory:**

    ```sh
    ‚ùØ cd TabKeeper
    ```

3. **Install the dependencies:**

	**Using [npm](https://www.npmjs.com/):**

	```sh
	‚ùØ npm install
	```

4. **Build:**

	Build the project with:

	**Using [npm](https://www.npmjs.com/):**

	```sh
	npm run build
	```

5. **Load the Extension in Chrome:**
	1. Open Chrome and go to chrome://extensions/.

	2. Enable Developer mode (toggle in the top right).

	3. Click "Load unpacked".

	4. Select the dist folder generated by the build step.

	5. Your extension is now installed and ready to use!

6. **Development:**

	Develop the project with:

	**Using [npm](https://www.npmjs.com/):**

	```sh
	npm run build:watch
	```

	This command will continuously watch your source files and rebuild the extension automatically when changes are detected.
